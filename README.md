# NFT Marketplace 🚀

A decentralized NFT marketplace built using:

- Solidity
- Remix IDE
- React.js
- IPFS (Pinata)

## Features
- Mint NFTs
- Upload metadata to IPFS
- View NFTs in UI
- List and buy NFTs

## How to Run
npm install
npm start
